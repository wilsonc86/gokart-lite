Three steps to embed the map widget for indicative burning program

1. Configure gokart options


var gokartOptions = [{
    app:"ibp", //the application name
    containerId:"map1" //the id of the html element which contains the map widget
    name:"gokartIbp" //the global variable name of this gokart instance, default is gokart_[app]
},{
    app:"todaysburns", //the application name
    containerId:"map2" //the id of the html element which contains the map widget
    name:"gokartTodaysburns" //the global variable name of this gokart instance, default is gokart_[app]
}]

2. Declare the html element to contain the map widget


<div id="map1" style="border-style:double;width:100%;height:600px" />
<div id="map2" style="border-style:double;width:100%;height:600px" />

3. Load gokart's bootstrap.js 


<script type="text/javascript" src="https://ssslite.dpaw.wa.gov.au/dist/static/js/bootstrap.js" ></script>
